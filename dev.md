# Guia para configurar e executar o projeto

## Comando para criar ambiente virtualizado na máquina local
- python -m venv venv

## Ativar ambiente virtualizado que foi criado
- Windows: venv\Scripts\activate
- Linux/Mac: source/bin/activate

## Instalar dependências
- pip install -r requirements.txt

## Comando para executar a aplicação
- python app.py
